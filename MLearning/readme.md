# Machine Learning Models

This folder showcases a project that uses data processing from a database to then apply a Machine Learning model that can predict results based on certain variables, as well as obtain the model's evaluation indicators.


📑 Content

- Link to Colab where the analysis can be viewed: https://colab.research.google.com/drive/1fE57pVEJ_gFvpUVjzfXx4ISDlqigNu1y?authuser=1

- Link to the folder containing Colab and the original database: https://drive.google.com/drive/folders/11JhB0WWQK6AFOR3cmgBtAIT4D5DSJB0b?usp=drive_link

🎯 Objective

The objective of this project is to evaluate where in Uruguay the highest number of crimes occur, as well as to assess whether there are any patterns among the victims of these crimes by analyzing the data with statistical correlations and descriptive graphs.
