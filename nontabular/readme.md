# Analysis of Non-Tabular Data

This folder presents an analysis of non-tabular data. It begins with data processing, followed by the analysis of variables in the form of graphs to evaluate behaviors. The reviews are standardized to create a simpler database for implementing a predictive model. Two different predictive models are then implemented, and their performance is compared using metrics designed for this purpose.

The objective of this work is to analyze different reviews of the film Harry Potter and the Philosopher's Stone. In addition, various techniques will be used to predict a score based on the comments in different reviews.
